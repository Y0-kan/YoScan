# 更新日志
OneForAll的所有值得注意的更改都将记录在此文件中。

OneForAll的更新日志格式基于[Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

OneForAll遵守[语义化版本格式](https://semver.org/)。

- 实现了配置插拔式设计
